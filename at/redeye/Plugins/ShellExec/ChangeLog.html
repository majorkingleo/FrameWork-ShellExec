Versionsänderungsprotokoll

Dieses Protokoll listet die Änderungen von Version zu Version im Plugin auf.

Version 1.0 (2010-09-17)

Erste Version des Plugins, basierend auf JDatePicker 1.3.2 mit folgenden Änderungen:
  • Deutsche Lokalisierung: Montag ist erster Tag der Woche.
  • Deutsche Übersetztungen.
  • Anpassung der Farben, damit die Texte auch unter Linux gut lesbar sind.
